# persian-sort

sort arrays by Persian alphabetic order

## Installing

```bash
npm install persian-sort
```

Using yarn:

```bash
yarn add persian-sort
```

## Example

```js
const psort = require('persian-sort');
const sortedList = psort(list);
```
